# Changelog

All notable changes to OneStopShop will be documented in this file.

## [1.0.1] - 2026-01-17

### Fixed
- Updated interface versions to current game clients:
  - Classic Era: 11506 (1.15.6)
  - Cataclysm Classic: 40402 (4.4.2)

### Added
- MIT LICENSE file
- GitHub issue templates for bug reports and feature requests

## [1.0.0] - 2026-01-17

### Added
- Initial release
- Trade chat advertising with customizable templates and rate limiting
- Buyer detection for WTB/LF patterns in trade chat and whispers
- Party management with one-click invites and optional auto-kick
- Service tracking with session, daily, and all-time statistics
- Quick-cast spell buttons using SecureActionButtonTemplate
- Class-aware UI (Mage portals vs Warlock summons)
- Minimap button with drag-to-reposition
- Slash commands (`/oss`, `/onestopshop`)
- Configuration panel for all settings
- Support for Classic Era, TBC Classic, and WotLK Classic
